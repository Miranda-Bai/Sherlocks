**Install iat:**
`sudo apt-get install iat`

**Convert the .bin file to .iso file:**
`iat source.bin target.iso`

**Mount the .iso file:**
`sudo mkdir -p /mnt/disc1`
`sudo mount -o loop target.iso /mnt/disc1

**Unmount if you finish:**
`sudo umount /mnt/disc1/
